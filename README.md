# Personal Homelab Setup

Created this to quickly restore my raspberrypi in the event of a full restore

## Services

- Docker
- Tailscale
- Nginx
- Glances (Monitoring)
- Dozzle (Container logs)

HTTPS certs provided by Tailscale. Nginx being used as a reverse proxy to route traffic to appropriate containers

Homelab can currently be accessed by devices on my tailnet
